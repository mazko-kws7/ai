Technical Analysis

Lets calculate some technical indicators on both datasets:

  • Moving Averages (20-day and 50-day for daily, 20-hour and 50-hour for hourly)
  • Relative Strength Index (RSI) (14-day and 14-hour)
  • Moving Average Convergence Divergence (MACD)
# Technical Indicators
def calculate_indicators(df, period1, period2):
    # Moving Averages
    df['MA_Short'] = df['Close'].rolling(window=period1).mean()
    df['MA_Long'] = df['Close'].rolling(window=period2).mean()

    # RSI
    delta = df['Close'].diff()
    up, down = delta.copy(), delta.copy()
    up[up < 0] = 0
    down[down > 0] = 0

    avg_gain = up.rolling(window=14).mean()
    avg_loss = abs(down.rolling(window=14).mean())

    rs = avg_gain / avg_loss
    df['RSI'] = 100 - (100 / (1 + rs))

    # MACD
    exp12 = df['Close'].ewm(span=12, adjust=False).mean()
    exp26 = df['Close'].ewm(span=26, adjust=False).mean()
    df['MACD'] = exp12 - exp26
    df['Signal'] = df['MACD'].ewm(span=9, adjust=False).mean()
    return df
