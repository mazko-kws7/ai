Calculations for RSI (Relative Strength Index)

To calculate the RSI, I need to determine average gains and average losses over a period. Since I don’t have the full historical data, I’ll use a simplified approach focusing on the provided data. A typical RSI period is 14.

Daily Chart RSI Approximation:

  1. Calculate Daily Changes: Determine the price change for each day (Close - Previous Close).
  2. Separate Gains and Losses: List the gains (positive changes) and losses (negative changes).
  3. Calculate Average Gain and Loss: Average the gains and losses over a shorter recent window (e.g., the last 10-14 days) to get a sense of recent momentum.
  4. Calculate RS: RS = Average Gain / Average Loss
  5. Calculate RSI: RSI = 100 - (100 / (1 + RS))

Disclaimer: The RSI calculation below is an approximation due to limited historical data. A true RSI would use a 14-period average, which is not feasible with the provided dataset.

def calculate_rsi(data, period=14):
    gains = []
    losses = []
    for i in range(1, len(data)):
        change = data[i] - data[i-1]
        if change > 0:
            gains.append(change)
        else:
            losses.append(abs(change))

    # Use a simplified average over the period
    avg_gain = sum(gains[-period:]) / period if len(gains) >= period else sum(gains) / len(gains) if gains else 0
    avg_loss = sum(losses[-period:]) / period if len(losses) >= period else sum(losses) / len(losses) if losses else 0

    if avg_loss == 0:
        return 100  # Avoid division by zero, indicates strong uptrend
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    return rsi
